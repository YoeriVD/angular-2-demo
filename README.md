# angular-2-demo

Instructions:
-------------

(found on http://angular.io )
install typings

    $ npm install -g tsd
    $ tsd query angular2 --action install
 install typescript and compile
 
    $ npm install -g typescript@^1.5.0-beta
    $ tsc --watch
